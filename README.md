### 세팅
1. 깃 클론

       git clone https://github.com/ksh1224/finstagram.git

2. 해당 프로젝트 모듈 설치

       yarn  

       or

       npm install

3. 환경 변수 설정 후 실행

       yarn start

       or

       npm run start
